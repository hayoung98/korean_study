    韓文字的書寫筆順


      原則1

      先寫子音，再在其右側或下方寫母音。

      如果只是單純想要表現母音「ㅏ」[a] 或「ㅣ」[i] 時，

      也要先寫子音「ㅇ」[ng]。


    ■ 「 子音 + 母音 」

      ● 母音寫在右側或下方
    ㅎ[h] + ㅏ[a] = 하[ha] ㄱ[k] + ㅜ[u] = 구[ku]

      ● 只有母音時，必須先寫「ㅇ」[ng]
    ㅇ[ng] + ㅣ[i] = 이[i] ㅇ[ng] + ㅛ[yo] = 요[yo]



      原則2

      承接上述「原則1」的說明，

      如果母音後接子音的情況，就要寫在母音下方。

      但有時也會出現兩個子音並列的情形。


    ■ 「 子音 + 母音 + 子音 」

      ● 「子音 + 母音」後再加子音時，必須擺在母音下方
    ㅎ[h] + ㅏ[a] + ㄴ[n]= 한[han] ㄱ[k] + ㅜ[u] + ㄱ[k]= 국[kuk]
     
    ㅇ[ng] + ㅓ[eo] + ㅂ[p] + ㅅ[s] = 없[eob]






    韓文字母



    單母音


    ㅏ ㅓ ㅗ ㅜ ㅡ
    a eo o u eu
    ㅣ ㅐ ㅔ ㅚ ㅟ
    i ae e oe wi

    對於單母音來說，不管話者發音的時間有多長，嘴唇和舌頭都不需要改變位置。


    單母音根據舌頭的位置，也就是舌位的高低和舌位的前後分成
    高母音、低母音、前母音、後母音。


    根據嘴唇的形狀分為圓唇母音和非圓唇母音。




    複合母音


    ㅑ ㅕ ㅛ ㅠ ㅒ ㅖ
    ya yeo yo yu yae ye
    ㅘ ㅙ ㅝ ㅞ ㅢ
    wa wae wo we ui

    在一個音節裡，兩個母音合在一起的音稱為複合母音。


    韓語中有 11 個複合母音。





    單子音


    ㄱ ㄷ ㅂ ㅈ ㅇ ㅋ ㅌ
    k/g t/d p/b z ng(無聲) k t
    ㅍ ㅊ ㅎ ㄴ ㄹ ㅁ ㅅ
    p ch h n r/l m s



    雙子音


    ㄲ ㄸ ㅃ ㅆ ㅉ
    g d b ss zz

    雙子音的發音跟單子音比起來較重。
